# 游记行文风格指南

## 核心原则

本指南定义了旅行游记的默认行文风格. 用户也可以指定一篇公开文章作为风格参考, 此时应动态抓取该文章并提取风格特征覆盖本指南.

## 默认风格: 朴素记录体

以下风格特征提炼自季长冰(知乎)的游记系列, 代表一种"不装"的旅行写作风格:

### 标点与格式
- 全文使用英文标点 (comma, period, colon, semicolon, parentheses)
- 不使用中文标点 (，。：；（）)
- 段落之间空一行
- 适当使用 Markdown 标题分层: 日期/城市作为二级标题, 景点作为三级标题

### 叙述方式
- 以时间线为主轴, 按日期和行程顺序推进
- 记录事实: 去了哪, 看了什么, 吃了什么, 花了多少, 怎么到的
- 直接陈述, 不展开抒情, 不发表人生感悟
- 可以有简短的主观评价, 但用词克制, 如 "感觉挺好", "值得一去", "比预期差一点"
- 实用信息自然穿插: 交通方式, 门票价格, 预约方式, 步行距离

### 句式特征
- 短句为主, 偶尔长句
- 不用排比句
- 不用比喻和修辞
- 不引用诗词或名人名言
- 口语化但不啰嗦, 像在和朋友聊天
- 偶尔夹带一点自嘲或幽默, 但不刻意搞笑

### 结构模板
```
## 前言
简述旅行背景, 为什么去, 大致行程概要, 花费概要.

## Day X: 日期 - 城市/地区
按时间顺序记录当天行程.
每个景点/活动一段或几段.
穿插实用信息.
配图(如有).

## 总结(可选)
简短总结, 几句话即可, 不升华.
```

### 绝对不要出现的内容
- "XX让我感受到了生命的意义"
- "站在XX面前, 我不禁想到..."
- "如果你也喜欢XX, 一定不要错过"
- "这座城市, 用它独特的方式..."
- 任何形式的心灵鸡汤
- 大段的历史科普(简短提及即可)
- "推荐指数: ★★★★★" 这类评分体系
